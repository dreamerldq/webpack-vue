# 前端架构

### 依赖下载
npm install

### 常用命令
npm run build 打包
npm run dev 启动开发环境    http://localhost:8080/html/***.html

### 编译器 vscode 配置
```
{
  "eslint.autoFixOnSave": true,
  "eslint.validate": [
      "javascript",{
          "language": "vue",
          "autoFix": true
      },"html",
      "vue"
  ]
}
```
